# Veri Dosyaları Yer Tutucusu

Bu klasör, ham veri dosyalarını içermelidir.

## Gerekli Dosyalar

Aşağıdaki dosyaları Kaggle'dan indirip bu klasöre yerleştirin:

1. `train.csv` - Eğitim veri seti
2. `test.csv` - Test veri seti (opsiyonel)

## İndirme Linki

[Kaggle - Airline Passenger Satisfaction](https://www.kaggle.com/datasets/teejmahal20/airline-passenger-satisfaction)

## Dosya Boyutları (Tahmini)

- `train.csv`: ~13 MB
- `test.csv`: ~3 MB
